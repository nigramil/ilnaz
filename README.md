# Random Password Generator

**Автор:** Мостафин Ильназ

## Описание

Программа позволяет генерировать случайные пароли с настройками длины и типа символов. Есть возможность сохранять историю сгенерированных паролей в файл и загружать её при запуске.

## Использование

1. Установите Python 3.x.
2. Запустите файл `password_generator.py`.
3. Настройте параметры:
   - Ползунок длины пароля.
   - Чекбоксы для выбора цифр, букв, спецсимволов.
4. Нажмите «Сгенерировать пароль».
5. Используйте кнопки для работы с историей.

## Пример теста

- Выставьте длину 16.
- Выберите цифры и буквы.
- Нажмите «Сгенерировать пароль».
- Проверьте, что пароль отображается и сохраняется в историю.
